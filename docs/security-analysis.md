# Security Analysis

## Controls Used
- WPA2-PSK for WiFi
- IPsec VPN for branch-to-branch traffic
- NAT for Internet access

## Threats Addressed
- Eavesdropping on WAN links
- Unauthorized wireless access
- Untrusted Internet transit

## Gaps / Future Enhancements
- Use WPA3 if supported
- Add ACLs to limit inter-branch services
- Add logging and NTP for audit
